# Changelog

## [1.0] - 2026-07-10

### Added
- Initial release
- AutoHotkey script for extracting tax decision data from Word files
- Support for both individual (cá nhân) and organization (tổ chức) types
- Support for multiple penalty types (Phạt tiền, Phạt cảnh cáo)
- GUI with folder selection
- Results display in ListView table
- Export to Excel functionality
- Export to CSV functionality
- Copy to clipboard functionality
- Batch processing for multiple files
- Progress tracking
- Status bar with real-time updates

### Features
- Automatic detection of person vs organization type
- Extraction of 20+ data fields
- User-friendly interface
- Easy copy and paste results
- Multiple export formats (Excel, CSV, Clipboard)
- Batch processing support
- Vietnamese language support

### Technical Details
- Built with AutoHotkey v1.1+
- Uses COM automation for Word document handling
- Supports .docx files
- Cross-platform clipboard support
